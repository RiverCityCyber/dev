Setting Up VS Code

Published on May 9th 2026 by Connor

There are a ton of options and extensions you can install for VS Code to make it the perfect coding platform for you. The following are just what I have found useful.

  • Prettier code formatter

    Install prettier, then run Cmd + Shift + P on MacOS to search. Type in " "[html]": {"editor.defaultFormatter": "esbenp.prettier-vscode", "editor.formatOnSave": true}". Save, and now when you save a HMTL file, Prettier will automatically format it for you. The same can be done without Prettier, but I enjoy how it formats.

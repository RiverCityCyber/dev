Raspberry Pi CAN-Bus & Dash-Cam System

CAN-Bus Reader v1 - Overview

I am currently developing the initial iteration of a microcomputer designed to interface with a vehicle's internal CAN bus for diagnostic logging and real-time data monitoring. The system will also support a USB webcam functioning as a dash camera, with video recorded to an onboard microSD card. Although sustained read/write operations are not ideal for microSD storage, this limitation is acceptable for an early prototype. Once the software is fully functional, the platform will be upgraded to a Raspberry Pi 5, enabling the use of an M.2 SSD for high-write video storage while reserving the microSD card exclusively for the operating system.

How to build your own Raspberry Pi Zero Dash-cam:

  1. Install Raspberry Pi OS Lite on a Raspberry Pi Zero
  2. Connect your USB camera of choice (I am using a Logitech Webcam)
  3. Update the system (sudo apt update && upgrade)
  4. Run v4l2-ctl --list-devices, this will show you your webcam's information

  5. Run the following code: ffmpeg \ -f v4l2 \ -input_format mjpeg \ -video_size 1280x720 \ -framerate 15 \ -i /dev/video0 \ -c:v copy \

Raspberry Pi CAN-Bus & Dash-Cam System

CAN-Bus Reader v1 - Overview

I am currently working on my initial itteration of a micro-computer that I can connect to my car's internal CAM-Bus system to record diagnostics. It will also have a USB webcam attached to work as a dash-cam, storing video on the onboard MicroSD card. While such strenuous read-writes to a MicroSD card are not advised, this is just the initial prototype. After the code works, I will be upgrading to a Raspberry Pi 5, for the ability to use M.2 SSDs for video storage and MicroSD for the OS.

How to build your own Raspberry Pi Zero Dash-cam:

  1. Install Raspberry Pi OS Lite on a Raspberry Pi Zero
  2. Connect your USB camera of choice (I am using a Logitech Webcam)
  3. Update the system (sudo apt update && upgrade)
  4. Run v4l2-ctl --list-devices, this will show you your webcam's information

  5. Run the following code: ffmpeg \ -f v4l2 \ -input_format mjpeg \ -video_size 1280x720 \ -framerate 15 \ -i /dev/video0 \ -c:v copy \
